About Us


While working closely with Northrop Grumman we've created a 3D Antenna Visualization Tool

Our AVT is currently a desktop application that allows users to visualize various 3D Models and have the ability to scale and rotate it via mouse movements.

Users also can load antenna locations via a CSV file that will place them on the specified model.

We currently have support for a few 3D file types such as .obj, .vtk, .stl, .ply and many more to come in the near future!
